# Personal Expense Dashboard (Excel)

## Project Overview
This project is an Excel-based Personal Expense Dashboard developed to analyze income and expenses. The dashboard provides insights into spending patterns and monthly financial performance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas

## Files Included
- Personal_Expense_Data.xlsx (Raw Dataset)
- Personal_Expense_Dashboard.xlsx (Data Cleaning, Pivot Tables, Dashboard)
- Dashboard_Screenshot.png

## Dashboard Features
- Total Income
- Total Expense
- Monthly Income vs Expense
- Category-wise Expenses
- Top 5 Expense Categories
- Income Sources
- Interactive Slicers

## Key Insights
- Identified highest expense categories.
- Compared monthly income and expenses.
- Tracked spending trends.
- Analyzed income sources.
